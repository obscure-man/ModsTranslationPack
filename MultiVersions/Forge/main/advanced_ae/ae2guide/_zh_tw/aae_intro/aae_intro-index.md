# AE 進階功能！

《AE 進階功能》模組著重於提升玩家使用 ME 系統的體驗，並擴展遊戲後期系統的潛力。

這款模組能讓你升級樣板供應器，使其能夠將物品推送至目標機器的特定一側。

你還可以搭建一座多方塊電腦，只要其擁有足夠的合成儲存空間，

就能夠同時進行無限數量的合成作業，並且這些作業可以共享電腦的協同處理器。

此外，模組還包含多項實用功能，能大幅提升你的遊戲體驗！
