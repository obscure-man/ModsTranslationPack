# ExtendedAE 的起點

<Row>
<BlockImage id="extendedae:machine_frame" scale="8"></BlockImage>
</Row>

在收集足夠的 <ItemLink id="extendedae:entro_crystal" /> 之後，你終於可以合成第一個擴充機器框架了。

這些框架是《ExtendedAE》的基礎，模組中大多數方塊類型的機器，都會需要使用這些框架作為材料。
